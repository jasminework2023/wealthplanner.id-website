## 🔧 CARA GANTI (SUPER DETAIL)

### **STEP 1: Ganti `calculator.jsx` (PALING PENTING)**

#### Di Claude Design:
1. Buka project wealthplanner.id
2. Cari folder: `src` → `screens`
3. Cari file: `calculator.jsx` (file ini sekarang udah ada di situ)
4. **Klik file calculator.jsx untuk buka**

#### Copy File Baru:
1. Lihat file **`calculator.jsx`** yang aku sediain di bawah (paling panjang)
2. **SELECT ALL** teks (Ctrl+A atau Cmd+A)
3. **COPY** (Ctrl+C atau Cmd+C)

#### Paste ke Claude Design:
1. Kembali ke Claude Design
2. **DELETE semua teks lama** di calculator.jsx (Ctrl+A → Delete)
3. **PASTE file baru** (Ctrl+V atau Cmd+V)
4. **SAVE** (Ctrl+S)
